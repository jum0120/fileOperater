# fileOperater
檔案處理系統:含檔案收集、批量改名、編號增加等功能
# 如何使用
執行"File Operater.exe"
# 功能總覽
1：showFiles（展示檔案及資料夾結構）

2：collectFiles（複製所有檔案到coll內）

3：collectFilesLink（複製所有檔案捷徑到coll內）

4：numberAdd（將已編號檔案調整編號）

5：renameFileByTxt（依.txt檔內容批量更改檔案名稱）
